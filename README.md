# AI Monitoring System (Sentinel)

Intelligent surveillance system with person tracking and ML-powered gesture detection.

## Quick Start
1. `cd /Users/hemanthchowdary/mv/ai-monitoring-system`
2. `python app.py`
3. Open http://127.0.0.1:5001

## Features
- Real-time CCTV simulation
- Multi-person tracking (YOLOv8)
- Hand landmark extraction (MediaPipe)
- Gesture verification engine
- Web-based dashboard
